## Production Tips

### Recording Setup

**Screen Recording:**
- Use OBS Studio or Loom (1080p, 60fps)
- Clean browser window (close unnecessary tabs)
- Disable notifications
- Use incognito mode
- Cursor highlighting enabled

**Voiceover Recording:**
- Quiet room with minimal echo
- USB microphone (Blue Yeti, Audio-Technica AT2020)
- Pop filter to reduce plosives
- Record in Audacity or Adobe Audition
- Speak clearly, moderate pace (140-160 words/minute)

**Editing Software:**
- DaVinci Resolve (free, professional)
- Adobe Premiere Pro (paid)
- Camtasia (easy for beginners)
- Final Cut Pro (Mac)

### Visual Enhancements

**Transitions:**
- Smooth fades between clips (0.5-1 second)
- Zoom-in for emphasis on key features
- Pan across dashboard to show details

**Text Overlays:**
- Use brand colors (purple #8b5cf6)
- Sans-serif font (Inter, Poppins, Montserrat)
- Animate text with subtle fade-ins
- Keep text on screen for 3-5 seconds minimum

**Cursor Effects:**
- Highlight clicks with subtle circles
- Smooth cursor movement (not jerky)
- Pause briefly on important elements

**Music:**
- Royalty-free from Epidemic Sound, Artlist, or YouTube Audio Library
- Upbeat, modern, tech-focused
- Volume: -20dB to -25dB (background)
- Fade in at start, fade out at end

## Voiceover Tone Guide

**Overall Tone:**
- Professional but approachable
- Enthusiastic without being salesy
- Empathetic to teacher challenges
- Confident in solution

**Vocal Delivery:**
- Clear enunciation
- Moderate pace (not too fast)

- Emphasize key words (AI, instant, personalized)
- Smile while recording (it shows in voice!)

**Avoid:**
- Monotone delivery
- Overly technical jargon
- Rushed speech
- Filler words (um, uh, like)
